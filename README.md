 🛒 Shopping List Chrome Extension

A Chrome extension that lets you save and categorize URLs from your browser tabs.

## Features
- Save the current tab's URL with a category label
- Categories: Books, Clothes, Shoes, Electronics, Pets, Health
- Items grouped by category with clickable links
- Delete individual items permanently
- Data persists after closing the browser

## How to Install Locally
1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer Mode**
4. Click **"Load unpacked"** and select the project folder
